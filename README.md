# Stock Chart Creator
 
This creates charts of the last 10 day of stock prices for the 5 stocks listed.
 
## Description
 
You can use this program to quickly get 10 days worth of stock closing prices and see them graphed on charts. It's a great way to begin to understand how to make graphs with Python
 
## Getting Started
 
### Dependencies
 
* yfinance
* numpy
* matlotlib
 
### Installing
 
* You can install from github
 
### Executing program
 
* Simple to run you just run python project1.py from the folder it is installed
